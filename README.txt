How to Use SolarCalculator
---------------------------

1. Start by downloading the GitHub package.

2. Download these python libraries: flask flask-cors requests (This can be done with the command pip install flask flask-cors requests on Windows commandprompt )

3. Run server.py using python (this can be done with the command python [file_directory]) (to close the server press CTRL + C)

4. Open the `index.html` file to run the program in your web browser.

5. Use the search bar to click or search for your desired location.

6. Select either "All" or a specific season to view the optimal angle for solar panel efficiency.

Enjoy optimizing your solar setup!
