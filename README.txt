SolarCalculator - README
=========================

How to Use SolarCalculator
---------------------------

1. Start by downloading the GitHub package.

2. Open the `index.html` file to run the program in your web browser.

3. Use the search bar to click or search for your desired location.

4. Select either "All" or a specific season to view the optimal angle for solar panel efficiency.

Enjoy optimizing your solar setup!
